# ProyectiFinal_Coder_House - Ruben Gonzalez Gerding

Buenas noches, hago entrega del proyecto final del curso.

Se trata de un Club con 3 actividades y cada una de ellas tienen formularios de inscripción, con ellas se puede,crear, almacenar ,editar, y eliminar la informacion

La carpeta Proyecto Final contiene todo lo relacionado con la administracion y la carpeta club contiene toda la aplicacion, Dentro de la carpeta Club se encuentran las carpetas y subcarpetas Templates/paginas, que contienen los archivos .html. Se encuentran dentro de la carpeta Club tambien los archivos agregados urls.py , forms.py cons sus respectivos codigos y funciones.

Logre hacer la herencia en la pagina, sin redireccionar a una pagina en blanco, asique se puede probar desde la misma página de inicio, y comprobar los datos cargados desde el admin.
